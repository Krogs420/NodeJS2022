Mostly the git that I have used are the add - commit - push commands. Its used to get the code that is being worked on, put into a repository that can contain the code so it can be shared with others.

Node.js is based on JavaScript, and is a JavaScript runtime environment. It has all the tools needed to run a program written in JavaScript.

When running a file in the terminal you would put ‘node app.js’ for example, and that would execute the file and run the program.

Neat extension that goes amazing with node is nodemon, which runs the code for you every time you make a change to it.

To get started you would run ‘npm init --y’ in the terminal to get your package.json file and node_modules, which will give you context for installing packages that can be used in node like express.
