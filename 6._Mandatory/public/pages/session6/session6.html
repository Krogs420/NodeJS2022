Week 6 - SSR

CSR & SSR

CSR:

All the files that are in the public folder are essentially client files. Client Side Rendering means generating the HTML components on the browser side, by executing Javascript code within the browser that manipulates the HTML DOM to build the HTML nodes.

SSR:

app.js, node_modules, README.md, package.json etc. are all serverside files. Server Side Rendering generating the HTML for a webpage on the server side.
